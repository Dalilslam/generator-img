## 📖 Описание проекта

Marketplace Ad Generator - это мощный инструмент для автоматизации создания рекламного контента. Система позволяет генерировать продающие тексты и привлекательные изображения для карточек товаров на различных маркетплейсах.

### Ключевые функции:

- 🤖 **AI-генерация контента**: Автоматическое создание рекламных текстов и изображений
- 📊 **Аналитика**: Отслеживание эффективности рекламных материалов
- 📅 **Планирование**: Автоматическая публикация по расписанию
- 🎨 **Управление контентом**: Организация, фильтрация и хранение сгенерированных материалов
- 🔄 **Мультиплощадочность**: Поддержка Wildberries, Ozon, Яндекс.Маркет

## ✨ Основные возможности

### 1. Генерация рекламного контента
- Создание уникальных рекламных текстов на основе УТП
- Генерация 4 вариантов изображений для A/B тестирования
- Поддержка загрузки фотографий товара до 5 шт.
- Настройка стиля и тона коммуникации

### 2. Управление изображениями
- Фильтрация по вкладкам: новые, история, избранное, архив
- Массовое удаление и управление
- Добавление в избранное
- Пагинация и поиск

### 3. Аналитика и статистика
- Графики просмотров и кликов (CTR)
- Временные ряды с различной гранулярностью (день/неделя/месяц)
- Отслеживание конверсий
- Сравнительный анализ эффективности

### 4. Безопасность
- JWT-аутентификация
- Ролевая модель доступа
- Защита от перебора паролей
- Rate limiting для API

## 🛠 Технологический стек

### Backend
- **Django 4.2** - веб-фреймворк
- **Django REST Framework** - REST API
- **PostgreSQL** - основная база данных
- **Redis** - кэширование и очереди задач
- **Celery** - асинхронные задачи

### AI/ML
- **OpenAI API** - генерация текстов (GPT-4)
- **Stable Diffusion** - генерация изображений
- **DALL-E** - альтернативная генерация изображений

### Документация
- **Swagger/OpenAPI 3.0** - интерактивная документация
- **ReDoc** - альтернативный просмотр API

### DevOps
- **Docker** - контейнеризация
- **GitHub Actions** - CI/CD
- **Sentry** - мониторинг ошибок

## 🚀 Установка и запуск

### Предварительные требования

- Python 3.11+
- PostgreSQL 15+
- Redis 7+
- Node.js 18+ (для фронтенда)

### Быстрый старт (локальная разработка)

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/yourusername/marketplace-ad-generator.git
cd marketplace-ad-generator

# 2. Создайте виртуальное окружение
python -m venv venv

# Для Windows:
venv\Scripts\activate
# Для Linux/Mac:
source venv/bin/activate

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Настройте переменные окружения
cp .env.example .env
# Отредактируйте .env файл под свои настройки

# 5. Создайте базу данных PostgreSQL
createdb marketplace_generator

# 6. Примените миграции
python manage.py makemigrations
python manage.py migrate

# 7. Создайте суперпользователя
python manage.py createsuperuser

# 8. Загрузите тестовые данные (опционально)
python create_test_data.py

# 9. Запустите сервер разработки
python manage.py runserver

# 10. В другом терминале запустите Celery
celery -A marketplace_generator worker -l info --pool=solo
