## 🚀 О проекте

**Marketplace Ad Generator** - это мощный инструмент для автоматического создания рекламных материалов с использованием искусственного интеллекта. Приложение генерирует продающие тексты и изображения для карточек товаров на популярных маркетплейсах.

### Ключевые возможности:

✨ **AI-генерация контента**
- Создание продающих рекламных текстов
- Генерация 4 уникальных изображений товара
- Поддержка разных ракурсов и стилей
- Адаптация под российские маркетплейсы

🎨 **Управление контентом**
- Просмотр истории генераций
- Система избранного
- Полноэкранный просмотр изображений
- Удаление и архивирование

📊 **Аналитика**
- Отслеживание просмотров и кликов
- Расчет CTR (Click-Through Rate)
- Визуализация данных в графиках
- Фильтрация по датам и периодам

## 💡 Функциональность

### 1. Аутентификация и авторизация
- Регистрация новых пользователей
- JWT-токены для безопасной аутентификации
- Защищенные эндпоинты API
- Управление сессиями

### 2. Генерация рекламного контента
- Ввод произвольного промпта
- Загрузка до 5 изображений товара
- Автоматическая генерация текста и изображений
- Поддержка нескольких AI-провайдеров

### 3. Управление изображениями
- Вкладки: "Новые", "История", "Избранное", "Архив"
- Пагинация результатов
- Массовое удаление
- Добавление/удаление из избранного

### 4. Аналитика и статистика
- Сводные метрики (просмотры, клики, конверсии)
- Временные ряды с группировкой по дням/неделям/месяцам
- Визуализация в виде графиков
- Экспорт данных

## 🛠 Технологический стек

### Backend
- **Django 4.2.7** - веб-фреймворк
- **Django REST Framework** - REST API
- **Simple JWT** - аутентификация
- **drf-spectacular** - OpenAPI/Swagger документация
- **SQLite** - база данных (для разработки)
- **Pillow** - обработка изображений

### Frontend
- **Vanilla JavaScript** - без фреймворков
- **CSS3** - анимации, градиенты, адаптивный дизайн
- **Font Awesome** - иконки
- **SPA (Single Page Application)** - динамическая маршрутизация

### AI Интеграции
- **Hugging Face API** - первичная генерация
- **Pollinations.ai** - резервная генерация
- **Stable Diffusion** - генерация изображений
- **GPT/FLAN-T5** - генерация текста

## 📦 Установка и запуск

### Предварительные требования

- Python 3.9 или выше
- pip (менеджер пакетов Python)
- Git (опционально)
- Hugging Face API ключ (опционально)

### Быстрый старт

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/Dalilslam/generator-img.git
cd marketplace-ad-generator

# 2. Создайте виртуальное окружение
python -m venv venv

# 3. Активируйте виртуальное окружение
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

# 4. Установите зависимости
pip install -r requirements.txt

# 5. Примените миграции
python manage.py makemigrations
python manage.py migrate

# 6. Создайте суперпользователя
python manage.py createsuperuser

# 7. Запустите сервер
python manage.py runserver
```

### Очень быстрый старт (через Docker)

Для быстрого запуска всего приложения (бэкенд + фронтенд) в изолированной среде выполните команду в корневой директории:

```bash
docker-compose up --build
```
